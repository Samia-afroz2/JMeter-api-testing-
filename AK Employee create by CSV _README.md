# JMeter API Testing Project

## Project Overview
This project contains API testing scripts created using Apache JMeter.
The main goal is to test the employee creation API with multiple users.

## Tools Used
- Apache JMeter 5.6.3
- REST API
- CSV Data Set Config

## Test Scenarios
- Create employee API (POST request)
- Parameterized data using CSV
- Multiple users testing using Thread Group

## How to Run the Test
1. Open Apache JMeter
2. Load the `employee_create.jmx` file
3. Update CSV file path if needed
4. Click Start (▶️) button
